## 📖 How to Use

### Upload Files
1. Click **📷 Upload Images**, **🎥 Upload Videos**, **📄 Upload Documents**, or **📦 Upload Files**
2. Select your files
3. Bebe reads and analyzes them instantly
4. Files appear in the sidebar

### Chat with Bebe
1. Type your message in the input field
2. Press Enter or click "Send Message"
3. Bebe responds instantly
4. Use voice input with 🎤 button

### Switch AI Forms
- Click any form button to change Bebe's personality
- Each form has unique response styles

### Analyze Files
1. Click **🔍 Analyze Files** for detailed analysis
2. Click **📊 Generate Report** for comprehensive report
3. Click **📂 View All Uploads** to see everything
